# Raw Logs

Raw outputs are direct model responses, script traces, and judge outputs generated automatically by `benchmarks/15turn/benchmark_15turn.py`.

Use raw logs for audit and reproducibility. They should preserve the benchmark execution as closely as possible and should not be edited to improve readability.

The script writes raw logs using its `RAW_LOG` path under `results/15turn/raw_logs/` with a timestamped filename.
