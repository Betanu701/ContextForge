# Processed Results

Processed results contain aggregated scores, derived metrics, and clean benchmark summaries.

The benchmark script produces timestamped JSON and Markdown reports in `results/15turn/processed/` with:

- Grand total scores
- Per-cycle scores
- Per-turn average scores
- Latency metrics
- Token totals
- Content-filter counts
- Judge verdicts and reasoning

These files are the clean output layer for paper writing and benchmark interpretation.
