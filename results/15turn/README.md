# 15-Turn Results

This directory separates local generated benchmark artifacts from curated published results.

## Structure

- `raw_logs/`: local raw model responses and judge traces generated by the script
- `processed/`: local timestamped JSON and Markdown summaries generated by the script
- `runs/`: named published benchmark runs that are safe to cite from the repository

## Raw vs Processed

Raw logs are direct benchmark output and should be kept close to what the script emitted. Processed files are derived summaries intended for easier inspection, comparison, or paper writing.

## Publishing Rule

Commit only actual completed benchmark results. Do not commit placeholder summaries, empty `not_run` JSON files, or fake raw logs. If a run is incomplete or incomparable, mention it in the published run notes instead of presenting it as a result.
